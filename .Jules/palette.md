## 2026-05-24 - Keyboard Accessibility on Div-based Radio Groups
**Learning:** When implementing custom keyboard accessibility on div-based radio groups (`role="radiogroup"`), you must explicitly handle Space and Enter keys for selection. Additionally, the Space key triggers page scrolling by default, so `e.preventDefault()` must be called to ensure a smooth UX. Querying DOM elements within the parent container (e.g., `box.querySelectorAll`) rather than `document.querySelectorAll` is crucial to avoid unintended side effects on other parts of the UI.
**Action:** Always add keyboard listeners (`tabindex="0"`, `keydown` handlers) to custom interactive elements, ensure `e.preventDefault()` is used for the Space key where appropriate, and scope DOM manipulations to their respective containers.
