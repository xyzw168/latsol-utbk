## 2025-01-28 - Keyboard Accessibility for Custom Divs
**Learning:** Implementing `role="radio"`, `tabindex="0"`, and `keydown` support on interactive `div` elements is a robust, low-code pattern for accessible quiz options.
**Action:** Replicate this `role`/`tabindex`/`keydown` pattern for any future custom interactive components (like cards or buttons built with `div`s) rather than relying solely on `onclick`.
